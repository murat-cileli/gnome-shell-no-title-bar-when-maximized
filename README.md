**About**

Small utility that removes title bar when a window maximized in Gnome Shell.

**Requires**

- xdotool
- xorg-xprop
- xorg-xdpyinfo

**Usage**

Start script in background with:

    nohup ./no-title-bar.sh > /dev/null 2>&1&
